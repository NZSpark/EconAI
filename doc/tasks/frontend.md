# M9: 前端 SPA 任务清单

> 目录：`frontend/` | 技术栈：React 19 + TypeScript 5 + Ant Design / Shadcn

## 任务列表

### 项目初始化
- [x] M9-01 使用 Vite 创建 React + TypeScript 项目，配置 tsconfig、eslint、prettier
- [x] M9-02 安装 UI 组件库（Ant Design 或 Shadcn），配置主题/国际化（中文）
- [x] M9-03 配置路由（React Router）：登录页、项目列表、项目详情（子路由：知识库/任务）、管理页
- [x] M9-04 配置 API 代理（Vite proxy → api-gateway），开发环境 CORS 处理

### API 客户端层
- [x] M9-05 实现 axios/fetch 封装：baseURL、超时、统一错误拦截、请求 ID header
- [x] M9-06 实现 Auth Context：token 存储（localStorage 或 httpOnly cookie 由后端控制）、登录状态管理
- [x] M9-07 实现 token 自动刷新拦截器：401 → refresh → 重试原请求（避免并发刷新）
- [x] M9-08 实现请求 loading/error 状态管理 hook（useRequest）

### 认证页面
- [x] M9-09 实现登录页面：用户名 + 密码 + 登录按钮，错误提示（401/网络错误）
- [x] M9-10 实现登出功能：清除 token → 跳转登录页
- [x] M9-11 实现路由守卫：未登录 → 重定向登录页；角色权限不足 → 提示无权限
